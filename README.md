# 🚨 CityAlert

CityAlert est une application web permettant aux utilisateurs de signaler des incidents urbains (problèmes, dangers, anomalies) en temps réel.

---

## ⚙️ Stack technique

* **Frontend** : Vue.js + Vite
* **Backend** : Node.js + Express.js
* **Base de données** : MySQL
* **Authentification** : JWT
* **Upload fichiers** : Multer

---

## 📌 Fonctionnalités

* 🔐 Authentification utilisateur
* 📸 Ajout de signalements avec image
* 📍 Consultation des incidents
* 🛠️ Interface administrateur
